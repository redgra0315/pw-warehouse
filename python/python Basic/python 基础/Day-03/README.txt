python的基础数据类型
    集合 set
        集合的定位类型为：{}
        集合的特性：
            1、无序的
            2、不重复
        集合支持的函数：
            len() 判断长度
            in  not in  判断语句
        集合运算：
            {1,2,3,4,5,6} - {3,4}   差集
            {1,2,3,4,5,6} & {3,4}   交集
            {1,2,3,4,5,6} | {3,4,7} 并集
            {1,2,3,4,5,6} ^ {3,4,7} 对称差异
        定义空的集合：
            set()

    字典 {}
        字典的key必须是唯一的，字典有键值对组成
        key的特性：
            可以为字符串和数字
        value特性：
            value可以为任意的类型
        字典排序：
            sorted()
        定义字典：
            {}
