Socker 介绍

Socker套接字

Python中提供socker.py标准库，非常底层的接口库。
Socker是一种通用的网络编程接口，和网络层次额米有一一对应的关系。

协议族
--------------------
- AF_INET4 - IPV4  -
- AF_INET6 - IPV6  -
--------------------

Socker类型
-----------------------------------------
- SOCK_STREAM  面向联机的流套接字， TCP协议
- SOCK_DGRAM   无连接的数据报文套接字，UDP协议
