字符串、列表、元组
    
    列表的学习与练习
        列表为： []  
        索引方式访问
        索引的值是从0开始
        索引值大于等于0是从左到右算起，索引值小于0是从右到左算起
        如果索引值大于实际索引的最大值引发错误
        列表函数介绍：
            list.append(x)   将值添加到列表的尾部
            list.extend()    将列表整体插入至列表 L 中。
                注：使用加号(+)的方式并不会更改两个原始列表，而 extend 方法会修改调用列表的值。
            list + list      前后顺序会影响新列表的排列顺序。
            list.sort(reverse=False)  默认升序，可以通过改变 reverse 的值更改为降序
    元组的学习与练习
        元组为：()
        元组和列表相比的优点：
            元组比列表操作速度快；
            不允许修改，使数据更安全。
        元组和列表的操作方法都一样，列表是可变的，元组是不可变的
        但是元组可以转化为列表


    str list tuple 都是序列

    序列
        序列共有的操作,每个里面都是有自己的序号
            'hello world' [2]
            [1,2,3,4]     [2]
        切片
            [1,2,3,4,5] [0:3]
            [1,2,3,4,5] [-1:]
            "hello  world"  [0:8:2]
        判断语句
            in   not 
        len() 查看列表获取元组字符串的长度
        max() 查看列表获取元组字符串的最大值
        min() 查看列表获取元组字符串的最小值，如果字符串中有空格，最小的为空格
        ord() 字符串转换为asikema
   
    字符串
        字符串常用方法介绍：
        请参考list_3文件进行对比
        strip()  删除字符串开头、结尾处的空白符
        lower()  将字符串小写
        upper()  将字符串大写
        count()  返回 sub 在 S 中出现的次数
        startswith()  判断字符串是否以什么开头
        endswith()    判断字符串是否以什么结尾
        isdigit()     判断字符串是否是数字形式
        split()       按照 sep 分隔符切分字符串 S，seq 默认为空格。
        join()        将 S 字符串作为分隔符拼接 iterable 中的每个元素(元素必须为字符串)
        replace()     将字符串中的 old 字符串替换为 new。
        title()       返回字符串的标题版本，其中单词以大写字符开头，其余字符为小写。
